# otto

Otto is a cross-platform Electron desktop app (Windows/Mac/Linux) that acts as a general-purpose computer coworking agent, emphasizing action over guidance. It's user-invoked via global hotkey, built on the Claude Agent SDK (leveraging @claude-cdk packages), and combines computer-use (screenshot + mouse/keyboard), per-OS shell/process adapters, web search with page reading, and long-running observation tools to diagnose and fix problems like game stutters, misbehaving apps, or general system issues. Autonomy is configurable across three modes (strict / balanced / full allow) with every tool tagged by action class (read/reversible/destructive/irreversible), plan-step confirmations, a hard denylist for catastrophic commands, and a per-machine local markdown knowledge file that accumulates learned quirks across sessions.
